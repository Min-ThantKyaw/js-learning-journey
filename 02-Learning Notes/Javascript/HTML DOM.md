## append()
- element တစ်ခုရဲ့ last childနောက်မှာ one or several nodes or strings တွေကို appendလုပ်ဖို့သုံးတယ်။
- node objects နဲ့ strings နှစ်မျိုးလုံး acceptလုပ်တယ်
- append() မှာ return value မရှိဘူး
## appendChild()
- node objects တွေကိုပဲ acceptလုပ်တယ်
- append လုပ်လိုက်တဲ့ node objectကို return ပြန်ပေးတယ်
